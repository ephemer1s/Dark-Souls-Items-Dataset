            Gold-Hemmed Black Set is an Armor Set in Dark Souls and Dark Souls Remastered.

             

            Gold-Hemmed Black Set Information

            Worn by the witch Quelana of Izalith, mother of pyromancy and Daughter of Chaos.
            She wore this gold-hemmed black set before even the Age of Fire,
            and it offer strong resistance versus fire, poison, and other perils.

             

            Location

            • This set of armor can be obtained in the Demon's Ruin. Head towards the boss, Ceaseless Discharge, and go to the end of the path. You will find a grave with a corpse lying on top. Be careful, after picking it up the boss will start attacking you.

             

            Notes

            • Gold-Hemmed Black Set cannot be upgraded.
            • This set has the second highest Poison resistance in the game (with Painting Guardian Set being the first).
