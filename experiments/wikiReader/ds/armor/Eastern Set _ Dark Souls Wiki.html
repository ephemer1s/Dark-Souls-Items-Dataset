            Eastern Set is an Armor Set in Dark Souls and Dark Souls Remastered.

             

            Eastern Set Information

            Exquisitely crafted, this helm offers excellent defense, particulary versus slashing attacks, which are one of the main threats in battles in the East.
            Distinctive armor made in an Eastern Land. Exquisitely crafted, this armor offers excellent defense, particularly versus slash attacks, which are one of the main threats in battles in the East.

             

            Location

            • Darkroot Garden/Basin. This set of armor is located on a cliff on the far side of the sealed door in Darkroot Garden. Once the doors have been opened, follow the right wall until it terminates at a cliff. Follow the cliff to find the item perched on an outcropping.
              • Video Location Timestamp 12:53

             

            Notes

            • Eastern Set can be upgraded by using Twinkling Titanite.
            • Eastern Armor (chestpiece) reduces Stamina regeneration by 1 per second.
